Links

Este exemplo e sobre links e imagens.

Esta pagina em particular e sobre links! Temos tres tipos de link:

  • Link Absoluto, e como Pagina Extras
  • Link Relativo
  • Link Raiz-Relativo

Link absoluto e o caminho mais detalhado incluindo o esquema(protocolo cliente servidor utilizando ) o domio do site e seu caminho interno
Exemplo HTTPS://(Esquema) DEVELOPER.MOZILLA.OR(Dominio) /EN-US/DOCS/WEB/HTML(Caminho)

Link Relativo e o caminho que sub entende que o esquema e o dominio são o mesmo, precisando fornecer apenas o caminho exemplo "href='misc/extras.html'", lembrando que tem que colocar o caminho completo caso tenha pastas tanto acima como abaixo

Raiz relativo link e parecido com o link relativo porem sempre voltando para a pasta raiz do site(a camada mais acima)

Link Targets e utilizado para mudar o comportamento do elemento "a", exemplo por padrão os navegadores subistituem o conteudo de uma pagina pela pagina nova. Quando se usa o atributo target pode abrir em uma nova guia ou janela

Convenções de nomes

Como boa pratica utilizar o nome dos arquivos e pastas em minusculo alem de não deixar espaçoes em brancos nos nomes, pois não são permitidos espaços na URL, o navegador para contornar os espaços acaba adicionadno %20(codificação usada para representar espaço) utilizando hifem ao inves do espaço, sempre buscar um nome adequado para arquivos e pastas.

Imagens

Imagens são recursos externos ao HTML e para utlizar precisamos informar seu caminho para que o html possa renderizar, informando o caminho podendo ser absoluto, relativo ou raiz relativo URL. Utilizando a tag <img/> e o atributo src

Formato de Imagens

Tem quatro grandes padrões de imagens que são mais utilizados na web, cada um com vantagens e desvantagens.

imagem JPG

Utilizado para uma imagem com grande paleta de cores bem como arquivo compacto, utilizado geralmente para fotos e imagens com muito gradiente na imagem

GIF

Gifs são boas opções para animações simples, porem com uma paleta de cores reduzido.

PNG

PNGs são bons para tudo que não seja uma foto ou animação pois para a qualidade de fotos o PNG e JPG (pela percepição do olho humano) são iguais com o fato do PNG ser um arquivo maior, sendo utilizado para icones, diagramas, logos, etc.

SVG

SVG e baseado em vetor ou seja independente da escala utilizada ira manter a qualidade da imagem, sempre utilizar quando possivel

Tamanho da Imagem

Por padrão a tag <img/> deixa o tamanho de 150x150 pixels para JPG,GIF e PNG e SVG de 75x75pixels, porem pode ser alterado para qual tamanho desejar. Com os atributos de width e heigt, porem e recomendavel alterar as dimensões com CSS

Texto alternativo

Adicionando o atributo 'alt' a tag <img/> e uma boa pratica fornecendo um texto para a imagel tendo um impacto tanto nos motores de busca quando no usuarios que dependem exclusivamente de texto( pessoas com deficiência visual por exemplo)

Mais atributos em HTML

Linguagem do documento <html> pode ser utilizado <html lang='en'> para definir o idioma da pagina

Set de caracteres

seria como o alfabeto digital que a pagina pode entender e renderizar de forma correta

Poderia utilizr o UTF-8 que contem o idioma turco

  1. bir

  3. üç

<meta charset='UTF-8'> e responsavel por renderizar de forma correta outros idiomas

Para utilizar os caracteres reservados < > and & alem de outros deve se utilizar o & seguido de ponto e virgula ; e o codigo do caracter Lista de caracteres
