Listas

E desta forma que se faz uma lista não ordenada:

  • Adicione um "ul" elemento
  • Adicione cada item dentro com seu "li" elemento
  • Não e necessario uma ordem em particular

Já uma lista ordenada se parece assim:

  1. Utilizamos o novo "ol" elemtento envolta
  2. Porem, o "li" permanece o mesmo
  3. Note que os marcadores vão aumentando sozinhos
  4. Seguindo a ordem de uma lista ordenada
