Elemento HTML

Neste capítulo foram apresentados as tag que compõem um site sendo o <!DOCTYPE html> para mostrar ao navegador que esta e uma pagina em HTML 5 alem de sua estrutura como <Head> com informações para o navegador utilizar e <Body> onde estara o conteudo do site visivel ao usuario.

A tag <title> utilizada para mostrar o nome do site na guia do navegador.

A tag <h1> <h2>...<h6> são os títulos do site e a tag <p> utilizada nos paragrafos.

As tag para criar listas <ul>(lista não ordenada) e <ol>(lista ordenada) bem como <li> utilizada criar itens dentro das listas.

Elementos de ênfase como itálico <em> e negrito<strong>

Alem de diferenciar elementos em bloco que criar um espaço como a tag <p> e elementos em linha como <em>

A tag <br> e <hr> são tags que criam quebras de linhas.

Estrutura vs Apresentação

As tags de ênfase devem ser consideradas como marcação semânticas e deixar as marcações visuais para o CSS pois podemos alterar a cor de um texto sua fonte tamanho via CSS
